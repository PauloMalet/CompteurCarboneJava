Voila le projet de Paul Malet et de Barbara Dufour Bonan.

Pour l'executer, il suffit de Run la classe utilisateur.Utilisateur.
Il s'affichera alors des consignes dans la console et il est préférable de les suivre pour une bonne expérience utilisateur !
NB : rentrer les valeurs décimales avec une virgule et non un point.

Après avoir rentré toutes les valeurs, le bilan carbone s'affichera.
